Свёрстай меню кафе с помощью таблицы

На уроке мы сделали таблицу «Employee Information». Теперь примени те же приёмы — объединение ячеек, заголовки и цветные ряды — чтобы повторить меню кафе ниже.

1

Что нужно сделать

  • Создай файл index.html и подключи к нему style.css.
  • Свёрстай таблицу из 4 столбцов: № · Блюдо · Категория · Цена.
  • Первая строка — название кафе на всю ширину (объедини 4 ячейки в одну).
  • Вторая строка — заголовки столбцов.
  • Заполни таблицу минимум 4 блюдами и раскрась ячейки, как на превью.

Подсказки

HTML Теги для разметки

<table> Контейнер всей таблицы. Внутри живут строки.
<tr> Строка таблицы (table row). Каждая строка — отдельный <tr>.
<th> Ячейка-заголовок. Текст в ней по умолчанию жирный и по центру.
<td> Обычная ячейка с данными (table data) — блюдо, цена и т.д.
colspan="4" Атрибут для ячейки: объединяет её на 4 столбца. Нужен для названия кафе сверху.
<thead> <tbody> Необязательно, но красиво: «шапка» и «тело» таблицы.

CSS Свойства для стилей

border-collapse: collapse; Убирает двойные линии между ячейками — рамки сливаются в одну.
border: 2px solid; Рамка ячеек: толщина, тип линии и цвет.
background-color: ...; Цвет фона ячейки. Можно по имени (red) или по коду (#2e7d32).
color: ...; Цвет текста внутри ячейки.
text-align: center; Выравнивание текста по горизонтали (center / left / right).
padding: 14px; Внутренние отступы — чтобы текст не прилипал к рамке.
font-weight: bold; Жирность текста. Пригодится для цены и заголовков.

Проверь себя перед сдачей

  • Таблица состоит из 4 столбцов и хотя бы 6 строк.
  • Название кафе объединено в одну ячейку через colspan.
  • Заголовки сделаны через <th>, а не <td>.
  • У ячеек есть рамки, отступы и разные цвета фона.
  • CSS вынесен в отдельный файл style.css.
